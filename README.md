# Flintoshop

Used technologies:

## Frontend
- HTML5
- Bootstrap
- JQuery
- AJAX

## Backend
- Nodejs ( Express js )
- Mongoodb ( Mongoose )
- Handlebars
- Lodash
- Stylus
- NPM modules (passportJS, multer, async)

## Additional
- Gulp + plugins
- Mocha + Chai + Sinon
- Eslint
